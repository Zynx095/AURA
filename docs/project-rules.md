# A.U.R.A Project Rules  

Project Name:
A.U.R.A — Autonomous Unified Recognition Assistant

Architecture:
Frontend and backend must remain completely separated.

Frontend:
React + Vite + TailwindCSS

Backend:
FastAPI + Python

AI Systems:
YOLOv5 + OpenAI APIs

Core Rules:

* Never mix frontend and backend logic
* Backend must communicate only through APIs
* All systems must remain modular
* Avoid monolithic files
* Use reusable components and services
* Maintain clean folder structure
* Never hardcode sensitive credentials
* Preserve API naming consistency
* Every major module should be independently maintainable

Folder Ownership:
Frontend folder:
Owned by frontend/UI systems only

Backend folder:
Owned by AI/backend systems only

Project Goals:

* Real-time object detection
* AI conversational assistant
* Analytics dashboard
* Alert system
* Scalable architecture
* Modern responsive UI
