# Frontend Rules

Frontend Framework:
React + Vite + TailwindCSS

Rules:

* ONLY modify frontend folder
* NEVER modify backend logic
* NEVER rename API routes
* Preserve API response structure
* Use modular React components
* Keep UI responsive
* Maintain futuristic dashboard aesthetic
* Use reusable UI sections
* Keep animations lightweight
* Avoid inline styling where possible

Allowed Areas:
frontend/src/components
frontend/src/pages
frontend/src/layouts
frontend/src/styles

Forbidden:
backend/
database/
AI logic
YOLO systems
