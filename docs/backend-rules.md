# Backend Rules

Backend Framework:
FastAPI + Python

Rules:

* ONLY modify backend folder
* Maintain modular architecture
* Separate APIs into dedicated routes
* Keep detection pipeline isolated
* Avoid frontend dependencies
* Use reusable services/modules
* Add comments to important logic
* Maintain consistent JSON responses
* Never hardcode secrets or API keys

Core Backend Modules:

* detection
* assistant
* analytics
* database
* api

Forbidden:
frontend/
UI logic
CSS/styling
